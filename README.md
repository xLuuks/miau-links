# Mi&Au Links

Landing page institucional da **Mi&Au Clínica Veterinária**, desenvolvida como um site estático com foco em apresentação de serviços, especialidades, equipe e contato.

Repositório: [xLuuks/miau-links](https://github.com/xLuuks/miau-links)

## Visão Geral

O projeto foi construído com HTML, CSS e JavaScript puro, sem dependências ou etapa de build.

Principais destaques:

- Página única com navegação por âncoras
- Seções de serviços, especialidades, equipe e contato
- Galeria visual para banho e tosa
- Lightbox para ampliar imagens
- Botões de contato com WhatsApp
- Estrutura simples para edição e publicação rápida

## Tecnologias

- `HTML5`
- `CSS3`
- `JavaScript`

## Como Executar Localmente

Como é um projeto estático, você pode abrir o arquivo `index.html` diretamente no navegador.

Se preferir rodar com um servidor local, uma opção simples é usar a extensão **Live Server** no VS Code.

## Como Editar

- Conteúdo principal: `index.html`
- Estilos da página: `styles.css`
- Interações da interface: `script.js`
- Imagens e arquivos visuais: `assets/`

## Publicação

O projeto está versionado no GitHub e pode ser publicado facilmente com **GitHub Pages**.

Fluxo básico para atualizar o repositório:

```powershell
git add .
git commit -m "Atualiza a landing page"
git push
```

## Objetivo

Apresentar a clínica de forma clara e visual, facilitando o contato com clientes e a divulgação dos serviços oferecidos.
